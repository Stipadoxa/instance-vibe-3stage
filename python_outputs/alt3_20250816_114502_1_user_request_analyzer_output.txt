## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To display comprehensive information about a used laptop for sale, allowing the user to decide whether to contact the seller.

**Required Information & Actions:**

* **Heading:** 2015 MacBook Pro 13" - Excellent Condition

* **Key Information Blocks:**
    * **Gallery:**  (Image carousel)
        * Image 1: High-quality photo of the laptop from the front.
        * Image 2: Close-up of the keyboard showing minimal wear.
        * Image 3: Photo of the laptop's underside showing the serial number (partially obscured).
        * Image 4: Photo showing the laptop's screen displaying a simple desktop background.
    * **Price:** $450
    * **Condition:** Excellent - Minor scratches on the bottom case, fully functional.
    * **Specifications:**
        * Processor: 2.7 GHz Intel Core i5
        * RAM: 8GB
        * Storage: 256GB SSD
        * Operating System: macOS Ventura (Upgradable)
    * **Seller Profile:**
        * Profile Picture: [Image of a user's profile picture]
        * Username: @TechSavvySeller
        * Rating: ⭐⭐⭐⭐ (4.8 stars)  (123 reviews)
        * Member Since: Oct 2018
    * **Description:**  "Lightly used 2015 MacBook Pro 13-inch.  Perfect working condition, battery health at 85%.  Comes with original charger.  Selling due to upgrade."
    * **Location:** San Francisco, CA


* **Primary Actions:**
    * Contact Seller (Button)

* **Secondary Actions:**
    * View Seller Profile (Button, links to seller's profile page)
    * Report Listing (Button, opens a reporting form)


* **Required Status Indicators:**
    * "Listed 2 days ago"


## STAGE 3: ADD NAVIGATION

**Top Navigation:**
