## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a mobile marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To display all relevant information about a specific used laptop listing, allowing the user to decide whether to contact the seller.

**Required Information & Actions:**

* **Heading:** 2015 MacBook Pro 13" - Space Gray - Excellent Condition

* **Key Information Blocks:**

    * **Images:** Carousel of at least 4 high-quality photos showing different angles of the laptop (including any imperfections clearly indicated).
    * **Price:** $450
    * **Condition:** Excellent (minor scratches on the bottom casing) - *This should link to a detailed condition explanation page.*
    * **Seller Profile:**
        * **Profile Picture:** [Image of Seller Profile Picture]
        * **Seller Name:** John Doe
        * **Seller Rating:** ⭐⭐⭐⭐⭐ (150+ reviews)
        * **Join Date:** June 2022
    * **Description:** "Lightly used 2015 MacBook Pro 13" in Space Gray. Excellent condition with minor scratches on the bottom.  Includes original charger.  Battery health is at 85%."
    * **Specifications:**
        * **Processor:** 2.7 GHz Intel Core i5
        * **RAM:** 8GB
        * **Storage:** 256GB SSD
        * **Operating System:** macOS Monterey (upgradeable to Ventura)


* **Primary Actions:**

    * **Contact Seller:**  Button leading to an in-app messaging system with the seller.

* **Secondary Actions:**

    * **Report Listing:**  A small, unobtrusive button to report inappropriate content or scams.

* **Required Status Indicators:**

    * **Listing Updated:** "Last updated: 2 hours ago" (dynamically updates)
    * **Shipping Information:** "Shipping from: San Francisco, CA" (if applicable, and dynamically updated)


## STAGE 3: ADD NAVIGATION

**Top Navigation:**

* **Back:**  Icon and text indicating navigation back to the previous listing screen.

**Bottom Navigation:**

* **Home:** Icon and text to return to the main listings page.
* **Search:** Icon and text to access the search functionality.
* **Saved:** Icon and text to view saved listings.
* **Profile:** Icon and text to go to the user's profile.

**Note:**  The absence of a "Cart" icon in the bottom navigation assumes this is a used goods marketplace, where a traditional shopping cart isn't the primary mechanism for purchase.  Contacting the seller and negotiating the transaction are central.
