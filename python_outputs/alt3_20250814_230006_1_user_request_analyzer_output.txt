## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To display comprehensive details of a used laptop listing, allowing the user to make an informed purchase decision and contact the seller.

**Required Information & Actions:**

* **Heading:** 2015 MacBook Pro - 13-inch - Space Gray - Excellent Condition

* **Key Information Blocks:**
    * **Product Images:** Carousel of at least 4 high-quality images showcasing different angles of the laptop (e.g., front, back, keyboard, ports).
    * **Price:** $450
    * **Condition Details:** Excellent. Minor scratches on the bottom casing. Fully functional. Original box included.
    * **Seller Profile:**  Avatar image of seller John Doe, Seller Rating: 4.8 (123 reviews),  Seller Name: John Doe, Joined: Oct 2018
    * **Description:**  "Lightly used 2015 MacBook Pro 13-inch in Space Gray.  Excellent condition with minor cosmetic wear.  Includes original box and charger.  Specs: 2.7 GHz Intel Core i5, 8GB RAM, 256GB SSD. Battery health: 85%."
    * **Specifications:**  (collapsible section)  Processor: 2.7 GHz Intel Core i5, RAM: 8GB, Storage: 256GB SSD, Screen Size: 13-inch, Graphics: Intel Iris Graphics 6100, Operating System: macOS Monterey


* **Primary & Secondary Actions:**
    * **Primary:** Contact Seller (Button)
    * **Secondary:** Add to Favorites (Button), Report Listing (Button)

* **Required Status Indicators:**
    * "Last updated: 1 hour ago" (Timestamp indicating the freshness of the listing)
    * "12 watchers" (Social proof)


## STAGE 3: ADD NAVIGATION

**For Mobile Apps:**

* **Top Navigation:**
    * Back (Icon: Left-facing arrow)

* **Bottom Navigation:**
    * Home (Icon: House)
    * Search (Icon: Magnifying glass)
    * Messages (Icon: Speech bubble)
    * Profile (Icon: Person)

**Empty State Considerations:**  If the seller hasn't uploaded images, display placeholder text "Images coming soon" along with the message "Contact the seller for more information."  If no reviews are available, display "No reviews yet". If the description is missing, display "Seller has not provided a description yet."
