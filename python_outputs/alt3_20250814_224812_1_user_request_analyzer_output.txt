## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To provide comprehensive information about a used laptop listing, allowing the user to make an informed decision about whether to contact the seller.

**Required Information & Actions:**

* **Heading:** 2015 MacBook Pro 13-inch - Excellent Condition

* **Key Information Blocks:**

    * **Product Images:**  Carousel of at least 4 high-quality images showing different angles of the laptop (e.g., open, closed, keyboard, ports).
    * **Price:** $450
    * **Condition Details:** Excellent. Minor scratches on the casing. Fully functional. Original box included.
    * **Seller Profile:**  Avatar image of seller John Doe; Seller Rating: ⭐⭐⭐⭐⭐ (5/5 stars) from 12 reviews; Seller name: John Doe; Joined: October 2020
    * **Description:** "Lightly used MacBook Pro 13-inch from 2015.  Excellent condition with minor cosmetic wear.  Runs smoothly and includes the original box and charger.  Specs: 8GB RAM, 256GB SSD, Intel Core i5."
    * **Specifications:**  (Bulleted list):
        * Processor: Intel Core i5
        * RAM: 8GB
        * Storage: 256GB SSD
        * Operating System: macOS Monterey
        * Battery Health: 85%


* **Primary & Secondary Actions:**

    * **Primary:** Contact Seller (Button)
    * **Secondary:**  Report Listing (Button)


* **Required Status Indicators:**

    * **Listing Status:**  "Posted: 2 hours ago"


## STAGE 3: ADD NAVIGATION

**Top Navigation:**

* Back (Icon: Left-pointing arrow)

**Bottom Navigation:**

* Home (Icon: House)
* Search (Icon: Magnifying glass)
* Messages (Icon: Speech bubble)
* Profile (Icon: Person)

**Content Examples for situations with no data:**

* **No Images:** Placeholder image indicating "No images available."
* **No Seller Rating:** "New seller" or "No ratings yet".
* **No Description:** "Seller has not provided a description yet."
