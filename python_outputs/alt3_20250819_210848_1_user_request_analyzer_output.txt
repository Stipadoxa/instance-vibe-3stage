## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To provide a comprehensive view of a used laptop listing, enabling the user to make an informed purchase decision.

**Required Information & Actions:**

* **Heading:** 2018 MacBook Pro 13" - Space Gray - Excellent Condition

* **Key Information Blocks:**

    * **Image Carousel:**  Four high-quality images showcasing the laptop from different angles. (Images would be provided by the seller).  Includes a zoom function.
    * **Price:** $850
    * **Condition:** Excellent (minor scratches on the bottom bezel)
    * **Seller Profile:**  Profile Picture (circular),  Seller Name: John Smith, Seller Rating: ⭐⭐⭐⭐⭐ (4.8/5, 123 Reviews)
    * **Description:** "Lightly used MacBook Pro 13" from 2018.  Space Gray color.  In excellent condition with minor scratches on the bottom bezel (pictured).  Includes original charger.  Specs: 16GB RAM, 512GB SSD, Intel Core i5."
    * **Specifications:**
        * **Processor:** Intel Core i5
        * **RAM:** 16GB
        * **Storage:** 512GB SSD
        * **Operating System:** macOS Ventura
    * **Shipping Information:** Free shipping within the continental US.


* **Primary & Secondary Actions:**

    * **Primary:**  "Contact Seller" button (opens chat window)
    * **Secondary:** "Add to Cart" button (if implemented, takes user to a cart view)


* **Required Status Indicators:**

    * "Last Updated: 1 hour ago" (timestamp)
    * "Item location: New York, NY" (optional, based on seller input)


## STAGE 3: ADD NAVIGATION

**Top Navigation:**

* **Left:**  "Back" icon (navigates back to previous listings)
* **Right:** "Share" icon (Opens a sharing dialog)


**Bottom Navigation:**

* **Home:** House icon
* **Search:** Magnifying glass icon
* **Cart:** Shopping cart icon (badge indicating number of items in the cart if applicable)
* **Profile:** Person icon
