## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a mobile marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To provide comprehensive information about a used laptop listing, allowing the user to make an informed purchasing decision and contact the seller.

**Required Information & Actions:**

* **Heading:**  2015 MacBook Pro 13-inch - Excellent Condition

* **Key Information Blocks:**

    * **Image Carousel:**  (Four high-quality images showing different angles of the laptop, including the keyboard and ports.  Arrows for navigation.)
    * **Price:** $450
    * **Condition:** Excellent (Minor scratches on the bottom casing)
    * **Seller Profile:**
        * **Profile Picture:** [Image of seller's profile picture]
        * **Seller Name:** John Doe
        * **Seller Rating:** ⭐⭐⭐⭐ (4.8 stars)  (123 reviews)
    * **Specifications:**
        * **Processor:** Intel Core i5
        * **RAM:** 8GB
        * **Storage:** 256GB SSD
        * **Operating System:** macOS Monterey
    * **Description:**  "Lightly used 2015 MacBook Pro 13-inch.  In excellent condition, with only minor scratches on the bottom casing.  Battery health is at 85%.  Includes original charger."
    * **Shipping Information:** Free shipping within the US.


* **Primary & Secondary Actions:**

    * **Primary Action:** Contact Seller (Button)
    * **Secondary Actions:** Add to Favorites (Heart Icon), Report Listing (Flag Icon)


* **Required Status Indicators:**

    * **Listing Updated:** Last updated: 1 hour ago



## STAGE 3: ADD NAVIGATION

**Mobile App Navigation:**

* **Top Navigation:**
    * Back (Left-aligned arrow icon)

* **Bottom Navigation:**
    * Home (House icon)
    * Search (Magnifying glass icon)
    * Messages (Speech bubble icon)
    * Profile (Person icon)
