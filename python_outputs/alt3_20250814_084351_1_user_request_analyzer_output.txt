## STAGE 1: UNDERSTAND THE REQUEST

1. This request belongs to the **E-commerce** domain.
2. The platform context is **mobile**.
3. The user wants to **view the details of a used laptop listing on a marketplace app to decide whether to purchase it.**


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose**: To display detailed information about a used laptop for sale, allowing the user to make an informed purchase decision and contact the seller.

**Required Information & Actions**:

* **Heading**: 2015 MacBook Pro 13" - Space Gray - Excellent Condition

* **Key Information Blocks**:
    * **Product Images:**  Carousel of at least 4 high-quality images showing different angles of the laptop (including the screen, keyboard, ports, and any noticeable wear).
    * **Price:** $450
    * **Condition:** Excellent (minor cosmetic scratches on the bottom casing)
    * **Seller Profile:**
        * **Profile Picture:** [Image of seller's profile picture]
        * **Seller Name:** John Doe
        * **Seller Rating:** 4.8 stars (123 ratings)
    * **Description:** "Used MacBook Pro 13-inch from 2015, in excellent condition.  Minor cosmetic scratches on the bottom, but otherwise in perfect working order.  Includes original charger.  Battery health is at 85%.  Specs: 2.7 GHz Intel Core i5, 8GB RAM, 256GB SSD."
    * **Specifications Table:**
        | Spec          | Value             |
        |---------------|--------------------|
        | Model         | MacBook Pro 13"    |
        | Year          | 2015              |
        | Processor     | 2.7 GHz Intel Core i5 |
        | RAM           | 8GB               |
        | Storage       | 256GB SSD          |
        | Battery Health| 85%               |

* **Primary & Secondary Actions**:
    * **Contact Seller:** Button to initiate a chat or messaging exchange with the seller.
    * **Add to Favorites (Heart Icon):**  Button to save the listing for later review.


* **Required Status Indicators**:
    * **Listing Updated:** "Last updated: 2 hours ago" (dynamically updates)
    * **Sold Out indicator:** (If applicable) "This item has been sold" displayed prominently.


## STAGE 3: ADD NAVIGATION

**Top Navigation:**

* **Back Arrow Icon:** Navigates back to the previous search results page.

**Bottom Navigation:**

* **Home Icon:** Navigates to the main marketplace home screen.
* **Search Icon:** Opens the search bar for finding other items.
* **Favorites Icon (Heart):**  Displays a list of saved listings.
* **Profile Icon:**  Opens the user's profile page.
