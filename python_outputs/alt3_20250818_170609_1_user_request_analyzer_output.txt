## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app to decide whether to purchase it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To provide comprehensive information about a used laptop for sale, enabling the user to make an informed purchase decision.

**Required Information & Actions:**

* **Heading:**  2018 MacBook Pro 13-inch - Space Gray - Excellent Condition

* **Key Information Blocks:**
    * **Image Carousel:**  Four high-quality images showcasing the laptop from different angles (showing the screen, keyboard, ports, etc.).
    * **Price:** $800
    * **Condition:** Excellent (minor scratches on the bottom casing)
    * **Seller Profile:**
        * **Profile Picture:** [Image of Seller's Profile Picture]
        * **Seller Name:** John Smith
        * **Seller Rating:** ⭐⭐⭐⭐ (4.8 stars)  (123 ratings)
    * **Description:** "Used MacBook Pro 13-inch, Space Gray, 2018 model.  Excellent condition with minor scratches on the bottom.  Includes original charger.  Specs: 16GB RAM, 512GB SSD, Intel Core i5."
    * **Specifications Table:**  A table detailing key specifications (Processor, RAM, Storage, Operating System, etc.).  Example:
        | Specification        | Value               |
        |----------------------|----------------------|
        | Processor            | Intel Core i5        |
        | RAM                  | 16GB                 |
        | Storage              | 512GB SSD            |
        | Operating System    | macOS Ventura        |
        | Battery Condition   | 85%                  |

* **Primary & Secondary Actions:**
    * **Primary:** Contact Seller (Button)
    * **Secondary:**  Add to Wishlist (Button)

* **Required Status Indicators:**
    * **Listing Updated:** "Last updated: 2 hours ago"


## STAGE 3: ADD NAVIGATION

**Top Navigation:**

* Back Button (Icon: Left-pointing arrow)

**Bottom Navigation:**

* Home (Icon: House)
* Search (Icon: Magnifying glass)
* Notifications (Icon: Bell)
* Profile (Icon: Person)
