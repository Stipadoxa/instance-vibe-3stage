## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view detailed information about a used laptop for sale, assess the seller's trustworthiness, and initiate contact to potentially purchase the item.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To display comprehensive details of a used laptop listing, enabling the user to make an informed decision about potential purchase.

**Required Information & Actions:**

* **Heading:**  2015 MacBook Pro 13-inch - Space Gray - Excellent Condition

* **Key Information Blocks:**

    * **Gallery:**  (Carousel of at least 4 high-quality photos of the laptop from various angles, including the screen, keyboard, and ports).  Example Photo captions: "Laptop Lid", "Keyboard and Trackpad", "Ports and Connections", "Overall Condition"
    * **Price:** $450
    * **Condition:** Excellent (Minor scratches on the bottom casing, fully functional)
    * **Seller Profile:**  (Includes profile picture, username:  "TechSavvySeller", Rating: 4.8 stars (12 reviews))
    * **Description:**  "Lightly used 2015 MacBook Pro 13-inch in Space Gray.  Excellent condition with minor cosmetic wear on the bottom.  Includes original charger.  Specs: 2.7 GHz Intel Core i5, 8GB RAM, 256GB SSD, macOS Ventura."
    * **Specifications:** (Table format):
        | Spec           | Value                   |
        |-----------------|--------------------------|
        | Processor       | 2.7 GHz Intel Core i5    |
        | RAM             | 8GB                      |
        | Storage         | 256GB SSD                |
        | Operating System| macOS Ventura            |
        | Condition       | Excellent                |


* **Primary Actions:**
    1. Contact Seller (Button)
    2. Add to Wishlist (Button)
    3. Report Listing (Button - less prominent)

* **Secondary Actions:**
    1. View Seller Profile (Link)

* **Required Status Indicators:**
    * "Last Updated: 2 hours ago"


## STAGE 3: ADD NAVIGATION

**Top Navigation:**

* Back (Icon: Left-pointing arrow)

**Bottom Navigation:**

* Home (Icon: House)
* Search (Icon: Magnifying glass)
* Wishlist (Icon: Heart)
* Profile (Icon: Person)

**Note:**  No data scenarios (e.g., no reviews for the seller) would be handled by displaying appropriate placeholders or messages, such as "No reviews yet" or a clear indication of a missing image.  Error handling for loading issues would also be implemented, but that is beyond the scope of the content specification.
