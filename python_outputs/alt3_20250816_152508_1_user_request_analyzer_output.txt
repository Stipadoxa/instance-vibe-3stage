## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To display comprehensive details of a used laptop listing, allowing the user to make an informed decision about purchasing.

**Required Information & Actions:**

* **Heading:** 2015 MacBook Pro 13-inch - Space Gray - Excellent Condition

* **Key Information Blocks:**
    * **Images:**  Carousel of at least 4 high-quality photos showing different angles of the laptop (including screen, keyboard, ports, and any signs of wear).
    * **Price:** $450
    * **Condition:** Excellent (Minor scratches on the bottom casing)
    * **Seller Profile:**
        * **Profile Picture:** [Image of seller's profile picture]
        * **Seller Name:** John Doe
        * **Seller Rating:** ⭐⭐⭐⭐ (4.8 stars)  (123 reviews)
    * **Description:** "Used MacBook Pro 13-inch from 2015. Space Gray.  Excellent condition with minor scratches on the bottom.  Runs smoothly.  Includes original charger.  Battery life is still good (approximately 6 hours)."
    * **Specifications:**
        * Processor: 2.7 GHz Intel Core i5
        * RAM: 8GB
        * Storage: 256GB SSD
        * Operating System: macOS Ventura


* **Primary Actions:**
    * **Contact Seller:**  Button that initiates a chat with the seller within the app.

* **Secondary Actions:**
    * **Report Listing:** Button to report inappropriate content or scams.
    * **Add to Favorites/Watchlist:** Button to save the listing for later.

* **Required Status Indicators:**
    * **Listing Updated:** "Last updated: 1 hour ago"


## STAGE 3: ADD NAVIGATION

**Top Navigation:**

* **Back:** (Icon: Left-pointing arrow) - Navigates back to the previous search results page.

**Bottom Navigation:**

* **Home:** (Icon: House) - Navigates to the main marketplace feed.
* **Search:** (Icon: Magnifying glass) - Opens the search bar.
* **Messages:** (Icon: Speech bubble) - Opens the messaging inbox.
* **Profile:** (Icon: Person) - Navigates to the user's profile page.
