## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To provide comprehensive details about a used laptop listing, allowing the user to make an informed purchase decision and contact the seller.

**Required Information & Actions:**

* **Heading:** 2015 MacBook Pro 13" - Space Gray - Excellent Condition

* **Key Information Blocks:**

    * **Image Carousel:**  (Four high-quality images of the laptop from different angles.  One image should be the main featured image.)
    * **Price:** $450
    * **Condition:** Excellent (Minor scratches on the bottom casing)
    * **Seller Profile:**
        * **Profile Picture:** [Image of seller's profile picture]
        * **Seller Name:**  John Doe
        * **Seller Rating:** 4.8 stars (123 reviews)
    * **Specifications:**
        * **Processor:** 2.7 GHz Intel Core i5
        * **RAM:** 8GB
        * **Storage:** 256GB SSD
        * **Operating System:** macOS Monterey
    * **Description:**  "Lightly used 2015 MacBook Pro 13" in Space Gray.  In excellent condition with minor scratches on the bottom.  Battery health is at 85%.  Comes with original charger."

* **Primary & Secondary Actions:**

    * **Primary:** Contact Seller (Button)
    * **Secondary:** Add to Favorites (Button/Icon)


* **Required Status Indicators:**

    * **Listing Updated:**  "Last updated: 2 hours ago"


## STAGE 3: ADD NAVIGATION

**Top Navigation:**

* Back (Icon and text)


**Bottom Navigation:**

* Home (Icon and text)
* Search (Icon and text)
* Messages (Icon and text)
* Profile (Icon and text)


**Note:**  No "cart" is explicitly included in the bottom navigation because the user request focused solely on the product detail page.  Adding a cart to the bottom navigation is reasonable but exceeds the explicit request.
