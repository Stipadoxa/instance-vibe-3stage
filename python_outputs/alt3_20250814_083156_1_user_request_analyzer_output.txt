## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** To view the details of a used laptop listing and potentially contact the seller to purchase it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To display all relevant information about a used laptop for sale, allowing the user to decide whether to contact the seller.

**Required Information & Actions:**

* **Heading:** 2015 MacBook Pro 13" - Excellent Condition

* **Key Information Blocks:**
    * **Image Carousel:**  (At least 3 high-quality images of the laptop from different angles.  Example images would be visually appealing and show the laptop's condition clearly).
    * **Price:** $450
    * **Condition:** Excellent (Minor scratches on the bottom case)
    * **Seller Profile:**
        * **Profile Picture:** [Image of Seller's profile picture]
        * **Username:** @TechSavvySeller
        * **Seller Rating:** 4.8 stars (123 reviews)
    * **Description:**  "Selling my 2015 MacBook Pro 13".  Excellent condition, always kept in a case.  Includes original charger.  Battery health is at 85%.  Runs macOS Ventura smoothly.  Reason for selling: Upgraded to a newer model."
    * **Specifications:**
        * Processor: 2.7 GHz Intel Core i5
        * RAM: 8GB
        * Storage: 256GB SSD


* **Primary & Secondary Actions:**
    * **Primary:** Contact Seller (Button)
    * **Secondary:** Add to Watchlist (Button)


* **Required Status Indicators:**
    * **Last Updated:** Updated 2 hours ago


## STAGE 3: ADD NAVIGATION

**For Mobile Apps:**

* **Top Navigation:**
    * Back (Icon: Left-pointing arrow)

* **Bottom Navigation:**
    * Home (Icon: House)
    * Search (Icon: Magnifying glass)
    * Listings (Icon: List)
    * Messages (Icon: Speech bubble)
    * Profile (Icon: Person)


**Note:**  If no images are available, a placeholder image indicating "No Images Available" should be displayed.  If the seller has no rating, the rating section should display "No Ratings Yet."  Likewise, if the seller provides no description, a message such as "Seller has not provided a description" should be shown.
