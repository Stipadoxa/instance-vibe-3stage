## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To display comprehensive details of a used laptop listing, allowing the user to make an informed decision about purchasing.

**Required Information & Actions:**

* **Heading:**  2015 MacBook Pro 13" - Excellent Condition

* **Key Information Blocks:**
    * **Images:** Carousel of at least 4 high-quality photos of the laptop (showing different angles, keyboard, ports etc.).
    * **Price:** $450
    * **Condition:** Excellent - Minor cosmetic wear on the bottom casing. Fully functional.
    * **Seller Profile:**
        * **Profile Picture:** [Display Seller's Profile Picture]
        * **Seller Name:** John Smith
        * **Seller Rating:** 4.8 stars (120 reviews)
    * **Description:** "Lightly used 2015 MacBook Pro 13-inch.  Excellent condition with minor cosmetic wear on the bottom.  Includes original charger.  Specs: 2.7 GHz Intel Core i5, 8GB RAM, 256GB SSD.  Battery health is at 85%."
    * **Specifications Table:** (In a clean, easy-to-read format)
        * Processor: 2.7 GHz Intel Core i5
        * RAM: 8GB
        * Storage: 256GB SSD
        * Battery Health: 85%
        * Operating System: macOS Ventura (Upgradable)

* **Primary Actions:**
    * **Contact Seller:** Button to initiate a chat or messaging interface with the seller.

* **Secondary Actions:**
    * **Report Listing:** (Small, discreet button)
    * **Share:** (Small, discreet button, possibly an icon)

* **Required Status Indicators:**
    * **Listing Updated:** "Last updated: 2 hours ago" (or similar, indicating freshness of information)
