## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain**: E-commerce
2. **Platform**: Mobile
3. **User Goal**: The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose**: To display comprehensive details of a used laptop listing, allowing the user to make an informed purchase decision and contact the seller.

**Required Information & Actions**:

* **Heading**: 2015 MacBook Pro 13" - Space Gray - Excellent Condition

* **Key Information Blocks**:
    * **Gallery:**  A carousel of at least 4 high-quality images of the laptop from different angles (showing keyboard, ports, screen etc.).
    * **Price:** $450
    * **Condition:** Excellent (Minor scratches on the bottom case)
    * **Seller Profile:**  
        * Profile Picture: [Image of Seller Profile Picture]
        * Seller Name:  John Doe
        * Seller Rating: ⭐⭐⭐⭐ (4.8/5 stars) - (123 reviews)
    * **Description:**  "This 2015 MacBook Pro 13" is in excellent condition. It has been meticulously cared for and comes with the original charger.  The battery life is still very good.  Only minor scratches on the bottom case are visible.  Ready to use."
    * **Specifications:**
        * Processor: 2.7 GHz Intel Core i5
        * Memory: 8GB RAM
        * Storage: 256GB SSD
        * Operating System: macOS Monterey


* **Primary Actions:**
    * Contact Seller (Button)

* **Secondary Actions:**
    * View Seller Profile (Button, links to seller's profile page)
    * Report Listing (Button, opens a reporting form)


* **Required Status Indicators:**
    * "Listed 2 days ago" (Timestamp of listing creation)


## STAGE 3: ADD NAVIGATION

**Top Navigation:**
* Back (Icon: Left-pointing arrow)

**Bottom Navigation:**
* Home (Icon: House)
* Search (Icon: Magnifying glass)
* Messages (Icon: Speech bubble)
* Profile (Icon: Person silhouette)
