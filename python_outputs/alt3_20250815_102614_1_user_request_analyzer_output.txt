## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To display all relevant information about a used laptop listing, allowing the user to decide whether to contact the seller.

**Required Information & Actions:**

* **Heading:** 2015 MacBook Pro 13" - Space Gray - Excellent Condition

* **Key Information Blocks:**

    * **Product Photos:**  A carousel of at least 4 high-quality images showing different angles of the laptop (including the keyboard, ports, and any blemishes).
    * **Price:** $450
    * **Condition:** Excellent (minor scratches on the bottom casing)
    * **Seller Profile:**
        * **Profile Picture:** [Image of seller's profile picture]
        * **Seller Name:** John Smith
        * **Seller Rating:** ⭐⭐⭐⭐⭐ (5/5, 123 reviews)
    * **Description:**  "Lightly used 2015 MacBook Pro 13" in Space Gray.  Excellent condition with minor cosmetic wear on the bottom.  Includes original charger.  Specs: 2.7 GHz Intel Core i5, 8GB RAM, 256GB SSD."
    * **Specifications Table:** (This is a table to clearly display specs. The content here is just an example and will vary based on the laptop model.)

| Spec          | Value              |
|---------------|----------------------|
| Processor     | 2.7 GHz Intel Core i5 |
| RAM           | 8GB                 |
| Storage       | 256GB SSD           |
| Operating System| macOS Monterey       |


* **Primary & Secondary Actions:**

    * **Primary:** Contact Seller (Button)
    * **Secondary:** Add to Wishlist (Button)


* **Required Status Indicators:**

    * **Listing Updated:** "Last updated: 1 hour ago"


## STAGE 3: ADD NAVIGATION

**Mobile App Navigation:**

* **Top Navigation:**
    * Back (Icon: Left-pointing arrow)
    * Share (Icon: Share icon)

* **Bottom Navigation:**
    * Home (Icon: House icon)
    * Search (Icon: Magnifying glass icon)
    * Messages (Icon: Speech bubble icon)
    * Profile (Icon: Person icon)

Note:  If no data is available for a field (e.g., seller rating), a placeholder message should be displayed (e.g., "No ratings yet" or "New seller").  The image carousels should display a "No images yet" message if no images are uploaded.
