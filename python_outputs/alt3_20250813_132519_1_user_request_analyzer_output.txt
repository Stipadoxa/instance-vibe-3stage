## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To provide comprehensive details of a used laptop listing, allowing the user to make an informed purchase decision and contact the seller.

**Required Information & Actions:**

* **Heading:** 2015 MacBook Pro 13-inch - Excellent Condition

* **Key Information Blocks:**

    * **Product Images:** Carousel of at least 4 high-quality images showcasing different angles of the laptop (e.g., front, back, keyboard, ports).
    * **Price:** $450
    * **Condition:** Excellent - Minor scratches on the bottom case. Fully functional.
    * **Seller Profile:**  
        * **Profile Picture:** [Image of seller's profile picture]
        * **Username:**  @TechSavvySeller
        * **Seller Rating:** ⭐⭐⭐⭐⭐ (4.8/5 stars) (123 ratings)
    * **Description:**  "Selling my well-maintained 2015 MacBook Pro 13-inch.  It's been used sparingly and is in excellent condition, with only minor cosmetic wear on the bottom.  It comes with the original charger.  Runs smoothly and has been regularly updated."
    * **Specifications:**
        * Processor: 2.7 GHz Intel Core i5
        * RAM: 8GB
        * Storage: 256GB SSD
        * Operating System: macOS Ventura

* **Primary Actions:**
    * **Contact Seller:** Button to initiate a chat or messaging conversation with the seller.

* **Secondary Actions:**
    * **Report Listing:** Button to report the listing if something seems amiss.


* **Required Status Indicators:**
    * **Listing Updated:** "Last updated: 2 hours ago" (dynamically updates)
    * **Shipping Information (if applicable):**  "Shipping from: California.  Shipping cost calculated at checkout." (or similar, depending on the seller's shipping setup)


## STAGE 3: ADD NAVIGATION

**Top Navigation:**

* **Back:**  Icon (left-aligned) to navigate back to the previous screen (search results or category page).

**Bottom Navigation:**

* **Home:** Icon (house icon) to go to the main app screen
* **Search:** Icon (magnifying glass icon) to access the app's search functionality
* **Cart:** Icon (shopping cart icon) to view the shopping cart
* **Profile:** Icon (person icon) to access the user's profile


**Handling No Data:**  If there are no images, a placeholder image will be displayed with a message like "No images available."  If the description is empty, a message like "No description provided" will be shown.  If the seller has no rating, the rating section will be absent.
