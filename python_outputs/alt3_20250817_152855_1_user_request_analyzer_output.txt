## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To provide comprehensive information about a used laptop for sale, allowing the user to make an informed decision about purchasing it.

**Required Information & Actions:**

* **Heading:**  2019 MacBook Pro 13" - Space Gray - Excellent Condition

* **Key Information Blocks:**
    * **Images:**  Carousel of at least 4 high-quality images showing different angles of the laptop (opened, closed, keyboard, ports).
    * **Price:** $850
    * **Condition:** Excellent - Minor scuffs on the bottom casing. Fully functional.
    * **Seller Profile:**
        * **Profile Picture:** [Image of seller's profile picture]
        * **Seller Name:**  John Smith
        * **Seller Rating:** ⭐⭐⭐⭐⭐ (5.0)  (123 Reviews)
    * **Description:** "Selling my lightly used MacBook Pro 13".  Great condition, always kept in a sleeve.  Includes original box and charger.  Specs: 16GB RAM, 512GB SSD, Intel Core i5."
    * **Specifications:**
        * **Brand:** Apple
        * **Model:** MacBook Pro 13" 2019
        * **Processor:** Intel Core i5
        * **RAM:** 16GB
        * **Storage:** 512GB SSD
        * **Condition:** Excellent

* **Primary & Secondary Actions:**
    * **Primary Action:** Contact Seller (Button)
    * **Secondary Action:** Add to Favorites (Button/Icon)


* **Required Status Indicators:**
    * **Last Updated:** Updated 1 hour ago


## STAGE 3: ADD NAVIGATION

**Mobile App Navigation:**

* **Top Navigation:**
    * Back (Icon: Left-pointing arrow)

* **Bottom Navigation:**

    * Search (Icon: Magnifying glass)
