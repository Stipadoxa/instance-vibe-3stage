## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To provide comprehensive information about a used laptop listing, enabling the user to make an informed purchase decision.

**Required Information & Actions:**

* **Heading:** 2015 MacBook Pro 13-inch - Excellent Condition

* **Key Information Blocks:**
    * **Images:**  Carousel of at least 4 high-quality images of the laptop from different angles (showing keyboard, screen, ports, etc.).
    * **Price:** $450
    * **Condition:** Excellent - Minor scratches on the casing. Fully functional. Battery health: 85%.
    * **Seller Profile:**
        * **Profile Picture:** [Image of seller profile picture]
        * **Seller Name:** John Smith
        * **Seller Rating:** ⭐⭐⭐⭐ (4.5 stars) - (123 reviews)
    * **Description:**  "Used MacBook Pro 13-inch from 2015.  Excellent condition with minor cosmetic wear.  Includes original charger.  Specs: 8GB RAM, 256GB SSD, Intel Core i5."
    * **Specifications:** (Table format)
        | Spec          | Value             |
        |---------------|--------------------|
        | Processor     | Intel Core i5     |
        | RAM           | 8GB               |
        | Storage       | 256GB SSD         |
        | Operating System| macOS Monterey    |


* **Primary & Secondary Actions:**
    * **Primary:** Contact Seller (Button)
    * **Secondary:** Add to Wishlist (Button), Report Listing (Button)


* **Required Status Indicators:**
    * "Listed 2 days ago" (Timestamp of listing creation)


## STAGE 3: ADD NAVIGATION

**Top Navigation:**
* Back (Icon: Left-pointing arrow)

**Bottom Navigation:**
* Home (Icon: House)
* Search (Icon: Magnifying glass)
* Notifications (Icon: Bell)
* Profile (Icon: Person silhouette)
