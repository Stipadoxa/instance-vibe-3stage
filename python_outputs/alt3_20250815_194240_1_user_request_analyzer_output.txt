## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** To view the details of a used laptop listing and potentially contact the seller to purchase it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To display comprehensive information about a used laptop for sale, allowing the user to decide whether to contact the seller.

**Required Information & Actions:**

* **Heading:** 2015 MacBook Pro 13" - Excellent Condition

* **Key Information Blocks:**
    * **Image Carousel:**  (Show at least 3 high-quality images of the laptop from different angles.  Include a clear indication of how many images are in the carousel.)  Example: (Carousel showing 5 images: image1.jpg, image2.jpg, image3.jpg, image4.jpg, image5.jpg)
    * **Price:** $450
    * **Condition:** Excellent - Minor scratches on the bottom case. Fully functional.
    * **Specifications:**
        * Processor: 2.7 GHz Intel Core i5
        * RAM: 8GB
        * Storage: 256GB SSD
        * Operating System: macOS Monterey (Upgradable)
    * **Seller Profile:**
        * Profile Picture: [Image of seller's profile picture]
        * Username:  @TechSavvySeller
        * Rating: 4.8 stars (12 reviews)
    * **Description:**  "Lightly used MacBook Pro 13-inch from 2015.  In excellent condition, with only minor cosmetic wear on the bottom.  Battery health is good (85%).  Includes original charger.  Runs smoothly and is ready for a new home!"


* **Primary & Secondary Actions:**
    * **Primary:** Contact Seller (Button)
    * **Secondary:** Add to Watchlist (Button)


* **Required Status Indicators:**
    * "Listed 3 days ago"


## STAGE 3: ADD NAVIGATION

**Top Navigation:**
* Back (Icon: Left-pointing arrow)


**Bottom Navigation:**
* Home (Icon: House)
* Search (Icon: Magnifying glass)
* Messages (Icon: Speech bubble)
* Profile (Icon: Person)

**(Note:  There is no need for a dedicated "Cart" icon in the bottom navigation since the focus is on a single product detail page.  The "Add to Watchlist" action addresses the user's need to save the item for later.)**
