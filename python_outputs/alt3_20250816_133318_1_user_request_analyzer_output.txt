## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** To view detailed information about a used laptop for sale, assess its condition and seller reliability, and initiate contact with the seller to potentially purchase it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To display all relevant information about a used laptop listing, enabling the user to make an informed decision about whether to contact the seller.

**Required Information & Actions:**

* **Heading:** 2015 MacBook Pro 13" - Excellent Condition

* **Key Information Blocks:**

    * **Gallery:**
        * Image 1: High-quality photo of the laptop's front
        * Image 2: High-quality photo of the laptop's side profile
        * Image 3: High-quality photo of the laptop's keyboard
        * Image 4: High-quality photo showing ports and connections.
        * Image 5: High-quality photo of the laptop's screen displaying a system information page (optional)

    * **Price:** $450
    * **Condition:** Excellent - Minor scuffs on the case, fully functional.
    * **Specifications:**
        * Processor: 2.7 GHz Intel Core i5
        * RAM: 8GB
        * Storage: 256GB SSD
        * Operating System: macOS Monterey
    * **Seller Profile:**
        * Profile Picture: [Image of seller's profile picture]
        * Username:  @TechSavvySarah
        * Rating: ⭐⭐⭐⭐⭐ (5/5)  (123 Reviews)
    * **Description:**  "Lightly used MacBook Pro in excellent condition.  Battery health at 85%.  Includes original charger.  Selling due to upgrade."


* **Primary & Secondary Actions:**

    * **Primary:** Contact Seller (Button)
    * **Secondary:** Add to Favorites (Button), Report Listing (Button)


* **Required Status Indicators:**

    * "Listed 3 days ago"
    * "Location: San Francisco, CA" (Optional:  Only if the seller allows their location to be shared)
