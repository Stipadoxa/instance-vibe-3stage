**STAGE 1 ANALYSIS:**
This is an e-commerce request for an unclear platform (could be mobile or desktop). The user wants to create a detailed product page for a used laptop listing on a marketplace to facilitate the sale of the laptop.


**STAGE 2 CORE CONTENT:**

*Screen Purpose*: To display comprehensive details of a used laptop, allowing potential buyers to review its features, condition, seller information, and facilitating contact with the seller.

*Primary Content*:
- Heading: "2020 Dell XPS 13 - Used"
- Key Information: Price: $650, Condition: "Good - Minor scratches on the bezel", Seller Rating: 4.8 stars (120 reviews),  Multiple high-quality images of the laptop (from various angles, showing keyboard, screen etc.),  Processor: Intel Core i5-1035G1, RAM: 16GB, Storage: 512GB SSD, Operating System: Windows 11 Home
- Primary Actions: "Contact Seller", "Add to Cart" (if applicable to the marketplace)


*Supporting Elements*:
- Secondary Actions: "Report Listing" (if inappropriate content), "Save Listing" (to favorites),  "View Seller Profile" (link to seller's page).
- Status Indicators: Listing ID: #12345,  "Posted: 3 days ago"
- Domain Elements: Seller's profile picture (with name and username) and a short seller bio (e.g., "Reliable seller, ships fast").  Detailed description of the laptop including any known defects or repairs. Mention of warranty if applicable.


*Content Examples*:
- Sample Data:  "This Dell XPS 13 is in good condition with minor cosmetic wear. The battery life is excellent, holding a charge for about 8 hours. It comes with the original charger.  The laptop was primarily used for office work."
- Microcopy:  "Contact Seller" button tooltip: "Send a message to the seller to ask questions or negotiate." "Add to Cart" button tooltip: "Add this item to your shopping cart." "Minor scratches on the bezel" - image highlighting the scratches.
- Empty State: (If the seller hasn't provided photos) "Seller is currently uploading photos, check back later." or a placeholder image.  If no description, "Seller has not provided a description."
