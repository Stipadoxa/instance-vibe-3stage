## Design Decisions
- **Layout Strategy**: Vertical layout for the main content area, allowing for a natural flow of information.  Header and Footer (implied for bottom navigation)
- **Component Choices**: Used system buttons (`10:3907`) for consistency, native text for flexibility in styling. Used a native `native-circle` component for the user avatar.
- **Hierarchy Decisions**: Primary action ("Contact Seller") is most prominent; secondary actions are less prominent and positioned logically based on their importance.
- **Gaps Addressed**: No carousel component in the design system.  Used horizontal scrolling using multiple `native-rectangle` components with `imageType: photo` for images.
