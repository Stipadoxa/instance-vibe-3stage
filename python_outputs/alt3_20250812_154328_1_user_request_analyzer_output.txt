**STAGE 1 ANALYSIS:**
This is an e-commerce request for a desktop/web platform. The user wants to create a detailed product page for a used laptop listing on a marketplace to facilitate a sale.


**STAGE 2 CORE CONTENT:**

*Screen Purpose*: To provide comprehensive information about a used laptop for sale, allowing potential buyers to make an informed purchase decision and contact the seller.

*Primary Content*:
- Heading: "2020 Dell XPS 13 - i7, 16GB RAM, 512GB SSD - Excellent Condition"
- Key Information: 
    - Price: $850
    - Condition: Excellent (Minor cosmetic wear on the bottom case)
    - Seller Rating: 4.8 stars (120 reviews)
    - Multiple Product Photos (Carousel of at least 4 high-quality images showing different angles and features of the laptop)
- Primary Actions: "Contact Seller", "Add to Watchlist"


*Supporting Elements*:
- Secondary Actions: View Seller Profile, Report Listing, Ask a Question (button with a small modal for typed questions)
- Status Indicators:  "Listing updated 1 hour ago"
- Domain Elements: Specifications (table with processor, RAM, storage, screen size, operating system, etc.),  Warranty Information (If any),  Shipping Details (or "Local Pickup Only"),  Payment Methods Accepted (PayPal, Credit Card)


*Content Examples*:
- Sample Data:
    - Specifications:  Processor: Intel Core i7-1065G7, RAM: 16GB LPDDR4X, Storage: 512GB NVMe SSD, Screen Size: 13.3", Operating System: Windows 11 Home.
    - Description:  "Lightly used Dell XPS 13 in excellent condition.  Minor cosmetic wear on the bottom case, but the screen and keyboard are flawless.  Battery life is excellent, runs all applications smoothly. Includes original charger."
- Microcopy:  "Contact the seller to ask any questions before purchasing."  "Watchlist this item to get notified of price drops."
- Empty State: (Not applicable in this context, a product page always displays product details, even if the seller hasn't provided all information.)

**STAGE 3 NAVIGATION:**

*Header Navigation*:  Home, Laptops, Electronics, My Account, Cart
*Breadcrumbs*:  Home > Laptops > Dell Laptops > 2020 Dell XPS 13 (This laptop listing page)
*Sidebar Navigation* (optional):  Filter by price, condition, brand, operating system, RAM, etc. (if this is part of a wider marketplace listing)
