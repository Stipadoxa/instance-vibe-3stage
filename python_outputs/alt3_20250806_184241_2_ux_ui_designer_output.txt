Sections are separated using `itemSpacing` within the main vertical container, and additional internal vertical containers separate sections to control vertical spacing.

7. **Visual Hierarchy:** Visual hierarchy is established by using larger text sizes for the title and price (`Title/Large`, `Headline/Medium`), and smaller text sizes for specifications and seller details (`Label/Medium`, `Caption/Small`). Images at the top establish visual prominence.
