## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To display comprehensive details of a used laptop listing, enabling the user to make an informed decision about purchase.

**Required Information & Actions:**

* **Heading:**  2018 MacBook Pro 13-inch - Space Gray - Excellent Condition

* **Key Information Blocks:**
    * **Images:** Carousel of at least 4 high-quality images showing different angles of the laptop (e.g., front, back, keyboard, ports).
    * **Price:** $850
    * **Condition:** Excellent. Minor scratches on the bottom casing. Fully functional.
    * **Seller Profile:**
        * **Profile Picture:** [Display a placeholder image if none is available]
        * **Seller Name:** John Smith
        * **Seller Rating:** ⭐⭐⭐⭐⭐ (5/5, 120 reviews)
    * **Specifications:**
        * Processor: 2.3 GHz Intel Core i5
        * Memory: 8GB RAM
        * Storage: 256GB SSD
        * Operating System: macOS Ventura
    * **Description:**  "Selling my well-maintained MacBook Pro.  It's been used primarily for light tasks like browsing and word processing.  Battery health is excellent. Comes with original charger."
    * **Shipping Information:**  Free shipping within the US.


* **Primary & Secondary Actions:**
    * **Primary:** Contact Seller (Button)
    * **Secondary:**  Add to Favorites (Button), Report Listing (Button, smaller, less prominent)


* **Required Status Indicators:**
    * "Listed 2 days ago"  (if applicable, adjust for time since listing)
    * "Last updated: 1 hour ago" (if applicable, dynamically updated)

**(Content Examples provided within the Key Information Blocks above)**

**(What to show when no data):**  If a seller has no rating, display "New Seller". If no description is provided, display "No description provided by seller."  If no images are provided, display a placeholder image indicating "No images available".
