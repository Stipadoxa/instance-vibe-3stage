## STAGE 1: UNDERSTAND THE REQUEST

1. **Domain:** E-commerce
2. **Platform:** Mobile
3. **User Goal:** The user wants to view the details of a used laptop listing on a marketplace app before potentially purchasing it.


## STAGE 2: GENERATE CORE CONTENT

**Main Screen Purpose:** To provide comprehensive information about a used laptop listing, allowing the user to make an informed decision about purchasing it.

**Required Information & Actions:**

* **Heading:**  2018 MacBook Pro 13" - Space Gray - Excellent Condition

* **Key Information Blocks:**
    * **Product Images:**  Carousel of 5-7 high-quality photos showing different angles of the laptop (including screen, keyboard, ports).
    * **Price:** $650
    * **Condition:** Excellent - Minor scratches on the bottom casing. Fully functional.
    * **Seller Profile:**
        * **Profile Picture:** [Display a profile picture of the seller]
        * **Username:**  @techsavvyseller
        * **Seller Rating:** 4.8 stars (123 reviews)
    * **Description:**  "Lightly used 2018 MacBook Pro 13" in Space Gray.  Excellent condition with minor cosmetic wear.  Includes original charger.  Specs: 16GB RAM, 512GB SSD, Intel Core i5.  Battery health: 90%."
    * **Specifications Table:**
        * **Brand:** Apple
        * **Model:** MacBook Pro 13"
        * **Year:** 2018
        * **Processor:** Intel Core i5
        * **RAM:** 16GB
        * **Storage:** 512GB SSD
        * **Color:** Space Gray


* **Primary & Secondary Actions:**
    * **Primary:** Contact Seller (Button)
    * **Secondary:** Add to Wishlist (Button)


* **Required Status Indicators:**
    * **Listing Updated:** Last updated: 2 hours ago


## STAGE 3: ADD NAVIGATION

**Top Navigation:**

* Back (Icon: Left-pointing arrow)

**Bottom Navigation:**

* Home (Icon: House)
* Search (Icon: Magnifying glass)
* Messages (Icon: Speech bubble)
* Profile (Icon: Person)

**(Note:  The "Add to Cart" button is omitted because the context suggests a "Contact Seller" approach is more appropriate for a used item marketplace.  A direct "Buy Now" button would also be inappropriate without seller confirmation.)**
